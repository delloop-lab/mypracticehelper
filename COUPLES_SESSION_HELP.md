# Booking shared sessions with multiple participants

A short guide for adding up to two extra participants to one appointment (up to 3 people total), so the same session appears in each participant's records.

## What's new

You can now add extra participants to a single appointment. This is useful for couples or family sessions: instead of creating separate appointments, you create one shared appointment with all participants on it.

If you only have one client in the appointment, nothing changes. Everything works exactly as before.

## When to use it

Use a shared appointment when everyone will attend the same session at the same time, for example:

- Couples therapy
- Two family members in the same session
- Any joint session where the notes and recording belong to both clients

If the two people are seen separately (different times), keep using two separate single-client appointments.

## How to add participants to an appointment

When creating a new appointment, or when editing an existing one:

1. Pick the first client like you always do (this is the primary client).
2. Choose one or two additional participants from the participant selectors.
3. Save the appointment.

That's it. The appointment now belongs to all selected participants.

You can only add participants who already exist in your client list. If someone is not there yet, add them as a client first, then come back and attach them to the appointment.

## How to recognise a shared session at a glance

You'll see clear signals so you're never in doubt:

- The appointment in your calendar/list shows combined names, for example: "Anna + John" or "Anna + John +1".
- A participant badge appears on the appointment when more than one person is attached.
- When you open the appointment, the title clearly shows all participants.
- Inside the session, a banner reminds you that notes and recordings are shared across all participants.
- On a single client's Sessions list, the row shows who else joined so you can identify shared sessions quickly.

## What gets shared between participants

For a shared appointment, the following are shared across all participants:

- The appointment itself (date, time, type, venue, duration).
- Session notes you write.
- Audio recordings and transcripts.
- Anything attached to the session.

Open the session from any participant's record and you'll see the same information. Anything you change in one place updates for everyone at the same time.

## What is NOT shared between participants

The following stay private to each client (as they always have):

- Each client's profile, contact details, intake forms, documents.
- Each client's other appointments.
- Each client's personal session history (only the joint sessions appear in both records, never the solo ones).

## Editing a shared appointment

When you edit a shared appointment:

- Whatever you change applies to all participants automatically (it's a single appointment, not copies).
- The app shows a warning that changes affect all participants.
- You can remove an additional participant at any time. After saving, that session is removed from that participant's records and remains with the others.

## Deleting a shared appointment

When you delete a shared appointment:

- The confirmation says the session will be removed from all participants' records.
- Confirming deletes the single shared appointment for everyone at once.

## Fees and payment

A shared appointment has one fee and one payment, not one per person. This is intentional: it's one session attended by multiple people.

- The fee/payment is tracked under the primary client (the first one you selected).
- Reminders for unpaid sessions appear under the primary client only, so you don't get reminded twice for the same session.
- Revenue isn't double-counted on the Payments page.

If you ever need to charge each person separately, create two separate single-client appointments instead.

## Single-client sessions are unchanged

If an appointment has only one client, nothing about it has changed:

- It looks the same.
- It only appears in that one client's record.
- Notes, recordings, fees, payments, reminders all behave exactly as they did before.

## Quick FAQ

- Q: I added someone by mistake. What do I do?
  - Edit the appointment and remove that participant, then save. The session remains with the other selected participants.

- Q: Can I add three or more people?
  - You can have up to 3 participants total in one session (1 primary + up to 2 additional participants).

- Q: Will the second client see the first client's name in their own profile?
  - Only on the joint session row (it shows "with {partner}"). Their profile, contact details, and other sessions remain private.

- Q: I started taking notes from one client's view. Will the partner's view see them?
  - Yes — the notes belong to the session, not to a single client. Both views show the same notes and recording.

- Q: Does this affect any existing appointments?
  - No. Existing appointments remain exactly as they were. The second-participant feature only kicks in for appointments where you explicitly add one.

If anything is unclear, contact your developer/support and they can walk through it with you.
